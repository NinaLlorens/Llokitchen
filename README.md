# Llo Kitchen

Application de gestion de stock alimentaire avec IA.

## Déploiement sur Vercel (une seule fois)

### 1. GitHub
1. Va sur github.com → "New repository"
2. Nomme-le `llo-kitchen`
3. Crée le repo (sans README)
4. Upload tous ces fichiers dedans

### 2. Vercel
1. Va sur vercel.com → "Add New Project"
2. Connecte ton repo GitHub `llo-kitchen`
3. Vercel détecte Vite automatiquement
4. Clique "Deploy"
5. Tu obtiens une URL fixe genre `llo-kitchen.vercel.app`

### 3. Sur iPhone
1. Ouvre l'URL dans Safari
2. Bouton partage ↑ → "Sur l'écran d'accueil"
3. C'est fait ! L'icône est installée pour toujours.

### Mises à jour
Quand on améliore l'app avec Claude :
1. Remplace le fichier `src/App.jsx` sur GitHub
2. Vercel redéploie automatiquement en 30 secondes
3. L'app sur ton iPhone se met à jour automatiquement au prochain chargement
**Pas besoin de toucher à l'icône sur l'écran d'accueil.**

## Développement local (optionnel)
```bash
npm install
npm run dev
```
